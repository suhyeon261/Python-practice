## 2장
### 02-1 숫자형
- 정수형: 정수를 뜻하는 자료형
- - a = 123
- - a = -178
- - a = 0

- 실수형: 소수점이 포함된 숫자
- 1) 일반적으로 볼 수 있는 실수형의 소수점 표현 방식
>> a = 1.2
>> a = -3.45
- 2) 컴퓨터식 지수 표현 방식
>> a = 4.24E10
>> a = 4.24e-10

- 8진수: 숫자가 0o 또는 0O(숫자 0 + 알파벳 소문자 o 또는 알파벳 대문자 O)로 시작하면 된다.
>> a = 0o177

- 16진수: 0x로 시작하면 된다.
>> a = 0x8ff
>> b = 0xABC

#### 숫자형을 활용하기 위한 연산자
- 사칙연산(+,-,*,/)
>> a = 3
>> b = 4
>> a + b ----> 7
>> a - b ----> -1
